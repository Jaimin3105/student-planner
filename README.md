# Smart Study Planner

A simple, beginner-friendly web application for managing your study schedule.

## Features
- User Authentication (Login/Register)
- Subject Management
- Task Tracking
- Automated Study Schedule Generation
- Dashboard with progress stats

## Tech Stack
- **Backend**: Python Flask
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Database**: SQLite

## How to Run
1. Ensure you have Python installed.
2. Install Flask:
   ```bash
   pip install flask
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser and go to `http://127.0.0.1:5000`
